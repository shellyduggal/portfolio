Case Study:
Mentor Program

The new Mentorship Program for the Capital Area Dental Society needed a better user experience. The page was difficult to read, only worked on large desktop browsers, and was a major burden on the webmaster for even the simplest of content updates.

I identified the problem and approached the webmaster with ideas for solutions. Utilizing the features of the previously installed BuddyPress plugin, I created a dynamic page with a modern, responsive design that updates automatically.

This project is currently live on the CADS website, but the Mentorship Program page is visible only to members. I have added some screenshots to show you the before and after.

The Challenge

With more dentists entering the workforce each year, local dental societies are searching for new ways to support their newest members. The Capital Area Dental Society recently launched a mentorship program to connect New Dentists with more experienced members in the group.

I visited the Mentorship Program page after it launched and had many concerns:

On a desktop it was difficult to read, on a mobile device it was impossible.

  • The mentors were listed in a spreadsheet embedded on the page
  • Multiple types of data (including images of various sizes) were being entered in each cell
  • Portions of the spreadsheet that extended past the viewport width were lost entirely (the page does not scroll horizontally)

I decided to reach out to the Webmaster and offer my web development services.

The Process

First, I made a simple design in HTML/CSS to use as a starting point to talk about the possibilities, the needs for the page, what we want it to look like and what features it should have.

Next, the webmaster and I sat down to explore the problem. In speaking with him, I learned of the difficulties he had in updating the page. It was time consuming to change the sheet then update the site every time a new mentor needed to be listed, or someone wanted their info to be changed.

We also discussed the overall website and his vision for it. He explained that the site was managed with WordPress and he had recently added bbPress and BuddyPress to help foster a sense of community among members.

I suggested we utilize the user profile feature of BuddyPress to create an entirely new mentor page that would allow mentors to update their own info.

We discussed project specifications then I set to work researching WordPress and plugins.

The Solutions

Create a child theme for the entire website to preserve customization and make future projects simpler

Use the XProfile feature of BuddyPress to add relevant mentor info fields to user profiles

Create a hidden "Mentors" group through BuddyPress, all mentors are members of this group and can only be added by invitation

Create a custom page template inside the child theme specifically for the mentorship program

Tweak the group members loop to pull selected profile data for each member and display it on the mentorship program page

Use the BuddyPress group members loop function to list all mentors on the mentorship program page template

The Result
  • Displays each mentor with a uniform size image, relevant info, and a link to their user profile
  • User profiles have been extended to include fields for all information present on the original mentorship spreadsheet
  • Allows mentors to update their own listings by simply updating their user profiles
  • Allows mentors to upload/crop their own photos
  • Fully responsive with modern design

This was a challenging and rewarding project in many ways.

This was my first experience with WordPress - it taught me the power, efficiency, and complexity of the service. WordPress does a beautiful job of making complex tasks easier for everyone from developer to client to customer.

I spent much of my time simply reading BuddyPress code to understand the features. I used the docs and forums as much as possible, but found my task was unique and I needed to understand the code to develop my own solutions.

This process taught me the importance of writing clean, maintainable code with good documentation, good comments, and reliable support. I am very thankful that the webmaster had chosen a quality plugin like BuddyPress!
